Base Color in ShaderGraph

The base color is the fundamental color of your material. It's like the "main" color that gets modified by lighting, textures, and other effects.

In ShaderGraph, you can set the base color using the Base Color property node.

// Example: Setting base color in a shader _BaseColor ("Base Color", Color) = (1,1,1,1)
